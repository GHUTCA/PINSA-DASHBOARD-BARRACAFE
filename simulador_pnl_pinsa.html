Cómo se calcula el EBITDA

El EBITDA es lo que queda en caja de la operación, antes de impuestos a la renta, intereses y depreciación. Se construye restando, en orden: los costos variables y los fijos a las ventas netas, y luego el IVA.
AVentas netas = Ventas brutas ÷ 1,19  (se quita el IVA del precio al público)
BTotal Costos Variables = COGS + Mermas + Costo Mercado Pago  (escalan con las ventas)
   MCU = A − B  (margen de contribución)
CTotal Costos Fijos = Remuneraciones + Imposiciones + Arriendo + Gasto Común + Gas/Agua/Energía + Otros
IVAa pagar = (A × 19%) − (COGS + Arriendo + G.Común + Gas/Agua/Energía + Otros) × 19%
EBITDA = A − B − C − IVA

Nivel de ventas donde EBITDA = 0, manteniendo los CLP fijos y constantes los %. "↑ falta X%" = cuánto deben subir las ventas para llegar a cero; "✓ cubierto" = ya está sobre el punto.
